# AI Memo App

Next.js 기반의 지능형 메모 관리 시스템입니다. AI 기술을 활용하여 사용자의 메모를 더욱 효율적으로 관리하고 활용할 수 있도록 도와줍니다.

## ✨ 주요 기능

- AI 기반 메모 분석 및 분류
- 직관적인 메모 작성 인터페이스
- 🔍 스마트 검색 및 필터링
- 📱 반응형 디자인
- 모던한 UI/UX
- ⚡ 빠른 성능과 최적화

## 🛠️ 기술 스택

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Development Tools**: [Taskmaster AI](https://github.com/taskmaster-ai/taskmaster-ai)
- **Package Manager**: npm

## 🚀 시작하기

### 필수 요구사항

- Node.js 18.17 이상
- npm 또는 yarn

### 설치 및 실행

1. **저장소 클론**

   ```bash
   git clone https://github.com/aq3aq4/ai-memo-app.git
   cd ai-memo-app
   ```

2. **의존성 설치**

   ```bash
   npm install
   ```

3. **개발 서버 실행**

   ```bash
   npm run dev
   ```

4. **브라우저에서 확인**
   [http://localhost:3000](http://localhost:3000)을 열어 애플리케이션을 확인하세요.
